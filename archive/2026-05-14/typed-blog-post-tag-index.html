Typed Blog Post Tag Index

You're building the content-discovery layer for a blogging platform. Raw post entries arrive as `unknown` from a CMS API; your engine must validate them, build a typed reverse index from tags to posts, and return a strongly-typed tag summary report — with zero `any`.

Goals

  • Define a generic discriminated union `ValidationResult<T>` with `ok: true / ok: false` variants.
  • Implement `validatePost` to narrow `unknown` → `BlogPost` through ordered field-by-field guards, returning a typed result.
  • Implement `buildTagIndex` to validate all raw entries, accumulate per-tag statistics in a single pass, and assemble a `TagIndexReport`.
  • Use `Pick`, `Record`, and tuple types from the provided interfaces — no `any`, no type assertions.
challenge.ts 

// Key types
export interface BlogPost {
  id: string;
  title: string;
  author: string;
  publishedAt: string;              // "YYYY-MM-DD"
  tags: [string, ...string[]];      // at least one tag
  wordCount: number;                // positive integer
}

export interface TagSummary {
  tag: string;
  postCount: number;
  avgWordCount: number;             // Math.round'd
  longestPost: Pick<BlogPost, "id" | "title" | "wordCount">;
  postIds: string[];                // sorted ascending
}

export type ValidationResult<T> = never; // TODO: discriminated union

// Main functions
export function validatePost(raw: unknown): ValidationResult<BlogPost>;
export function buildTagIndex(rawPosts: unknown[]): TagIndexReport;

Hints

  • A discriminated union is just two object types that share a literal-typed field — `ok: true` and `ok: false` are enough to let TypeScript narrow between them.
  • After checking `Array.isArray(tags) && tags.length > 0 && tags.every(t => ...)`, you can cast the result as the tuple type `[string, ...string[]]` — but only inside the guard where you've already proved it.
  • Use an intermediate accumulator object (e.g. `Record<string, { posts: BlogPost[]; totalWords: number }>`) to collect per-tag data before building each `TagSummary`.
