Typed Product Inventory Filter

You're building the catalog layer for an e-commerce storefront. Raw product entries arrive as `unknown` from a third-party supplier feed; your engine must validate them, apply typed filter criteria, and return a strongly-typed filtered inventory summary â€” with zero `any`.

Goals

  • Define `ProductFilters` with optional fields for category, stockStatus, maxPriceUsd, minRating, and tags.
  • Define `InventorySummary` with a filtered product array, total count, and an exhaustive per-Category breakdown.
  • Implement `parseProduct(raw: unknown): Product | null` that validates every field and returns null for any invalid input.
  • Implement `filterInventory` that discards invalid raws, applies all active filters, and returns a complete InventorySummary.
